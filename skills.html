My skills

Front-end

HTML

HTML is a markup language used for creating and structuring content on the web. It provides the foundation for building web pages by defining the structure and content of the page.

CSS

CSS (Cascading Style Sheets) is a language used for styling web pages. It allows for the separation of presentation and content, making it easier to maintain and modify the visual aspects of a website.

Javascript

JavaScript is a programming language used to create interactive web pages and web applications. It is primarily run on the client-side

Bootstrap

Bootstrap is a popular front-end framework used to create responsive and mobile-first web pages and applications quickly, using HTML, CSS, and JavaScript.

Backend-end

Node.js

Node.js is a JavaScript runtime built on the V8 engine, used for creating scalable and high-performance server-side applications, APIs, and command-line tools.

Express.js

Express is a fast and minimalistic web framework for Node.js that simplifies server-side web development with robust features and a flexible API.

React.js

React is a JavaScript library for building user interfaces. It uses a declarative approach to create reusable components and manage state, making it efficient and scalable.

Database

MySql

MySQL is an open-source relational database management system. It uses SQL to store, organize, and retrieve data, and is widely used for web applications and other data-driven projects.

MongoDb

MongoDB is a popular NoSQL document database that stores data in flexible, JSON-like documents. It is scalable, high-performing, and commonly used in web and mobile applications.
